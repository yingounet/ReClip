# ReClip

> 轻量、快速、隐私友好的 macOS 剪贴板历史管理器

## 特性

- 🔍 **快速搜索** - 模糊搜索文本内容和来源应用
- ⌨️ **键盘优先** - 完整的键盘导航支持（Vim 风格）
- 🔒 **隐私保护** - 自动过滤密码管理器内容
- 💾 **本地存储** - 所有数据存储在本地，不上传云端
- 🖼️ **多类型支持** - 文本、图片、文件
- ⭐ **收藏功能** - 收藏重要内容，永不自动清理
- 🎹 **全局热键** - ⌥⌘C 快速唤出

## 系统要求

- macOS 13.0 Ventura 或更高版本
- Apple Silicon 或 Intel 处理器

## 安装

### 从源码构建

```bash
# 克隆仓库
git clone https://github.com/yourusername/ReClip.git
cd ReClip

# 构建项目
swift build -c release

# 运行
.build/release/ReClip
```

## 使用方法

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `⌥⌘C` | 打开/关闭主窗口 |
| `↵` | 选中并粘贴 |
| `⌘↵` | 仅复制到剪贴板 |
| `⌫` | 删除选中项 |
| `⌘S` | 收藏/取消收藏 |
| `↑` `↓` | 上下导航 |
| `j` `k` | Vim 风格导航 |
| `⌘1-9` | 快速选择第 N 条 |
| `Esc` | 关闭窗口 |

## 隐私

- 所有剪贴板数据仅存储在本地
- 自动过滤密码管理器内容（1Password, Bitwarden 等）
- 不发起任何网络请求
